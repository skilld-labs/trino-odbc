All files are available in '/Library/ODBC/trino-odbc' after installation.

To setup a connection, you can use DSN to store your data source connection information,
1. Open 'iODBC Data Source Administrator'.
2. Go to 'User DSN' and click 'Add'.
3. Select 'Amazon Trino ODBC Driver' and click on 'Finish'.
4. Update the connection string values. For the list of all supported options, check '/Library/ODBC/trino-odbc/doc/connection-string.md' and '/Library/ODBC/trino-odbc/doc/macOS-dsn-configuration.md'.
5. Click 'Ok' to save changes.

For information on using the Amazon Trino ODBC Driver with Excel, see '/Library/ODBC/trino-odbc/doc/microsoft-excel.md'
